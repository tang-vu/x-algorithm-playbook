# Profile Optimization Checklist

> Optimize your profile for maximum conversion (profile visit → follow).

---

## Why Profile Matters

The algorithm tracks `profile_click` as a positive signal. When someone clicks your profile:

```
Profile visit → They decide to follow (or not) in seconds
```

Your profile is your conversion page.

---

## Profile Picture

### Requirements

- [ ] **Clear face shot** (or recognizable brand logo)
- [ ] **High resolution** (not blurry)
- [ ] **Good lighting** (professional look)
- [ ] **Consistent** with other platforms
- [ ] **Stands out** when small (timeline size)

### Testing

- [ ] Does it look good at 48x48 pixels?
- [ ] Is it recognizable?
- [ ] Does it represent my brand?

---

## Header/Banner Image

### Requirements

- [ ] **1500x500 pixels** optimal size
- [ ] **Communicates value proposition** or brand
- [ ] **Not text-heavy** (hard to read on mobile)
- [ ] **Visually appealing** and professional
- [ ] **Updated** (not outdated info)

### Ideas

| Type | Example |
|------|---------|
| Value prop | "Helping X achieve Y" |
| Social proof | Logos, numbers, testimonials |
| Personal brand | Signature style/colors |
| Featured work | Product shots, content preview |

---

## Bio (160 characters max)

### Structure Template

```
[What you do] + [Who you help] + [Unique value/credential]
```

### Requirements

- [ ] **Clear value proposition** in first line
- [ ] **Specific** (not generic)
- [ ] **Keyword-rich** for discoverability
- [ ] **Includes social proof** (if available)
- [ ] **Call to action** (optional)

### Examples

**❌ Bad:**
```
"Entrepreneur. Investor. Dreamer. Making the world better."
```

**✅ Good:**
```
"Helping SaaS founders hit $1M ARR | Built 3 exits | Daily growth tactics"
```

### Checklist

- [ ] Would someone know what I do in 3 seconds?
- [ ] Does it say who I help?
- [ ] Is there a reason to follow?
- [ ] Any social proof?

---

## Location

- [ ] Set to something relevant
- [ ] Can be location or tagline
- [ ] Not left blank

---

## Website Link

### Requirements

- [ ] Link works
- [ ] Points to highest-value destination
- [ ] Updated regularly

### Link Options (priority order)

1. Lead magnet / newsletter signup
2. Key resource for your audience
3. Personal site
4. Link-in-bio page (if multiple offers)

---

## Pinned Tweet

### This is CRITICAL

The pinned tweet is the first content people see. It should convert visitors to followers.

### Best Types of Pinned Tweets

| Type | When to Use |
|------|-------------|
| Best-performing thread | Showcase your best content |
| Value-packed resource | Attract your target audience |
| Introduction post | "Here's who I am and what I share" |
| Lead magnet | Convert followers to subscribers |
| Viral hit | Social proof of good content |

### Checklist

- [ ] Is this my best content?
- [ ] Does it represent what I post about?
- [ ] Does it have strong engagement?
- [ ] Updated in last 3 months?
- [ ] Would a new visitor want to follow after reading?

---

## Recent Posts

When someone visits your profile, they see your recent posts.

### Requirements

- [ ] Last 5-10 posts represent my niche
- [ ] Mix of content types
- [ ] Consistent quality
- [ ] Not all self-promotion
- [ ] Recent activity (not weeks old)

### Red Flags to Fix

- [ ] No posts in last week (looks abandoned)
- [ ] All promotional content (looks spammy)
- [ ] Off-topic posts (confuses visitors)
- [ ] Low engagement (social proof issue)

---

## Full Profile Audit Checklist

Run this monthly:

### Visual Elements
- [ ] Profile picture is optimal
- [ ] Banner is updated and effective
- [ ] Visual brand is consistent

### Text Elements
- [ ] Bio clearly communicates value
- [ ] Location is set
- [ ] Link is working and optimal

### Content Elements
- [ ] Pinned tweet is best content
- [ ] Recent posts represent my brand
- [ ] Activity is consistent

### Conversion Test
Ask yourself: If I landed on this profile for the first time, would I follow?

- [ ] Value is clear
- [ ] Expertise is evident
- [ ] Content looks interesting
- [ ] Professional appearance
